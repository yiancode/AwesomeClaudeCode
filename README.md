## 📖 关于本项目 | About This Project

**AwesomeClaudeCode** 是一个社区驱动的资源集合，汇集了 Claude Code 生态系统中最优质的资源、工具、教程和最佳实践。

**AwesomeClaudeCode** is a community-driven collection of high-quality resources, tools, tutorials, and best practices in the Claude Code ecosystem.

### ✨ 特色 | Features

- 🏛️ **官方资源优先** - 来自 Anthropic 的官方文档和工具
- 🌏 **双语支持** - 中英文内容并重，方便国际交流
- 🔄 **自动更新** - CSV 数据驱动，GitHub Actions 自动化维护
- ✅ **质量保证** - 所有资源经过验证和审核
- 📊 **分类清晰** - 13 个主分类，便于查找

### 🚀 快速开始 | Quick Start

1. 浏览 [目录](#-目录--contents) 找到感兴趣的分类
2. 点击资源链接访问详细内容
3. 查看 [官方资源](#-官方资源) 获取官方文档
4. 阅读 [贡献指南](CONTRIBUTING.md) 了解如何贡献
